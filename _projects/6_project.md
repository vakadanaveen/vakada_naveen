Final year undergraduate project — an autonomous robot capable of detecting and avoiding obstacles using regression-based navigation.

**Approach:**
- Used sensor data and regression models to predict steering corrections in real time
- Implemented on embedded hardware without relying on predefined obstacle maps

**Published:** ICCIDE 2020, Hyderabad, India.
